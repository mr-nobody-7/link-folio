# link-folio
A lightweight link tree clone

## Features
- Auth (signup/login with JWT + refresh tokens)
- Link management with drag-to-reorder
- Public profile with custom themes (6 themes)
- Analytics dashboard with click tracking
- QR code generation with branded download
- Open Graph meta tags for rich social sharing
- Visitor messages
- Rate limiting and Zod validation
- Temporary links with auto-expiry

## Tier 3 Verification
- Structural checks: `./scripts/tier3/verify-structure.sh`
- Runtime checks (with frontend/backend running): `./scripts/tier3/verify-runtime.sh`
